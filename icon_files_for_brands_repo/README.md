# RetroTINK Icon Files for Home Assistant Brands Repository

These icon files are ready to be submitted to the Home Assistant Brands repository.

## Files

- `icon.png` - 512x512px, square icon for the integration
- `icon@2x.png` - 1024x1024px, high-DPI version
- `logo.png` - 512x512px, same as icon (RetroTINK logo works well square)

## How to Submit

1. Fork https://github.com/home-assistant/brands
2. Create folder: `custom_integrations/retrotink/`
3. Upload these three files to that folder
4. Create a Pull Request with title: "Add RetroTINK-4K integration icons"
5. Wait for approval and merge

## File Specifications

✅ PNG format  
✅ Square aspect ratio (1:1)  
✅ Properly sized (512x512 and 1024x1024)  
✅ RGB color space  
✅ Black background (logo is white on black)  
✅ Properly trimmed  

## After Submission

Once your PR is merged:
- Icons will be available at `https://brands.home-assistant.io/retrotink/icon.png`
- All RetroTINK integration users will see the icon automatically
- May take 24-48 hours for caches to clear
